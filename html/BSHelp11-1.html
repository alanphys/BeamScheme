Base ParametersÂ¶

A variety of parameters are automatically calculated on the profiles. While these parameters can be reported directly they are more usually combined in an arithmetic expression. The available base parameters are:

  • Field Statistics
    • CAX Dose
    • Maximum Dose
    • Minimum Dose
    • Maximum position
    • Field Sum
    • Field Sum of Squares
    • Number of Profile Points
  • Interpolated Parameters
    • Left 90% dose distance
    • Right 90% dose distance
    • Left 80% dose distance
    • Right 80% dose distance
    • Left edge interpolated
    • Right edge interpolated
    • Left 20% dose distance
    • Right 20% dose distance
    • Left 10% dose distance
    • Right 10% dose distance
  • Inflection Point Parameters
    • Left inflection point
    • Right inflection point
    • 80% left inflection point distance
    • 80% right inflection point distance
    • 50% left inflection point distance
    • 50% right inflection point distance
    • 20% left inflection point distance
    • 20% right inflection point distance
    • Inflection point left slope
    • Inflection point right slope
  • Dose Point Parameters
    • Dose point 20% left
    • Dose point 20% right
    • Dose point 50% left
    • Dose point 50% right
    • Dose point 60% left
    • Dose point 60% right
    • Dose point 80% left
    • Dose point 80% right
  • Symmetry Base Parameters
    • Area left
    • Area right
    • Maximum dose ratio base parameter
    • Point difference base parameter
