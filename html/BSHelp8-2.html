The Image PaneÂ¶

The image pane displays the image, 2D array values, dose or fluence map. The image pane can be maximised in the application window by clicking the maximise button . The image can be windowed using the slider on the right hand side of the image. The image pane can be returned to its original state by clicking the minimise button . You can quickly move the profiles to an appropriate point on the image by clicking on this point with the mouse.

The image pane contains a tool bar with buttons that allow various operations can be carried out on the image. The File Open , Print PDF and Exit buttons are shortcuts to the equivalent menu bar functions. Other functions available are:

  • Invert

  • Normalise to CAX

  • Normalise to MAX

  • Centre Field
