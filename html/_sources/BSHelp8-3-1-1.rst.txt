File, Open
==========

Opens a dialogue box to select an image, dose map or 2D array file to load. See :ref:`Supported File Formats`

This function can also be accessed using the Open File button |open| on :ref:`The Image Pane` toolbar.

|Note| If you do not see the file you want make sure you have selected the correct file type.
