Photon Profile Parameters

The defined photon parameters are:

  • Field Centre
  • Field Size
  • Penumbra 80%-20%
  • Maximum Dose Ratio
  • Mean Value
  • Maximum Variation

Electron Profile Parameters

The defined electron parameters are:

  • Field Centre
  • Field Size
  • Penumbra 80%-20%
  • Maximum Dose Ratio
  • Flatness L90/L50
  • Maximum Variation

Flattened field

BeamScheme uses 80% of the field size as the flattened area.
